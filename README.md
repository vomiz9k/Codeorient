# Codeorient
Codeorient test task


Build:
```
bash build.sh
```

Run tests:
```
bash run_tests.sh
```

Usage: ```./tee [-a] <file>```

The tee command reads its standart input until end-of-file, writing a copy of the input to standart outputand to the file specified as argument.

Options:

-a:      Open file in append mode.

-h:      Show help.
